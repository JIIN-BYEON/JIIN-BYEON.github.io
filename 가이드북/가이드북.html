4. 모달 안에 PDF / 이미지 / 영상 넣기

상세 창 안에 발표 자료(PDF)를 띄우거나, 결과물 이미지를 보여주는 방법입니다.

방법 A. PDF(발표자료) 띄우기 (추천 ⭐)

프로젝트 데이터 안에 docUrl이라는 부분이 비어있을 겁니다. 여기에 PDF 경로를 적으면 모달 맨 아래에 문서 뷰어가 생깁니다.

  • 파일 위치: PDF 파일을 assets 폴더에 넣으세요.
  • 코드 수정:
// 수정 전
docUrl: "", 

// 수정 후
docUrl: "./assets/발표자료.pdf",

방법 B. 상세 내용에 이미지 넣기

modal-reflections (고찰) 부분에 이미지를 넣고 싶다면, HTML이 인식되도록 코드 한 줄을 고쳐야 합니다.

  1. 검색어(CTRL+F): modal-reflections" (따옴표 포함해서 검색)
  2. HTML 파일의 약 800번째 줄 쯤에 아래와 같은 코드가 있습니다.
    3. document.getElementById("modal-reflections").textContent = data.reflections;
  3. 이것을 아래처럼 .textContent를 .innerHTML로 바꾸세요.
    4. document.getElementById("modal-reflections").innerHTML = data.reflections;
  4. 이제 projectData 안의 reflections 부분에 <img> 태그를 쓸 수 있습니다.
    5. reflections: "내용 설명입니다... <br><br> <img src='./images/결과.jpg' style='width:100%;'>",

5. GitHub Pages 배포 팁

GitHub Repository를 만들고 파일을 업로드할 때, 반드시 폴더 구조를 유지해야 합니다.
  1. GitHub 저장소(Repository) 생성 (이름은 아이디.github.io 추천)
  2. index.html, guide.html, images 폴더, assets 폴더를 모두 업로드
  3. Settings > Pages 메뉴로 이동
  4. Branch를 main (또는 master) / /(root) 로 설정하고 Save
  5. 약 1~2분 후 https://아이디.github.io 주소로 접속 확인
